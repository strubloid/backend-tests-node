# Backend TypeScript Project

This project is a minimal backend setup using TypeScript. It includes:
- Basic TypeScript configuration
- Simple test setup for a TypeScript function
- Example for loading a text file

## Structure
- src/: Source code
- test/: Test files

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run tests:
   ```bash
   npm test
   ```

## Requirements
- Node.js
- npm

## Features
- TypeScript function testing
- File loading example
